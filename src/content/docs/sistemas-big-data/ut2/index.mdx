## Contenidos

-   Bases de datos SQL y NoSQL (MySQL, MongoDB, Cassandra).
-   Extracción e ingesta de datos con Python (pandas, PySpark, APIs, conectores).
-   Almacenamiento distribuido: HDFS y nube.
-   Procesamiento en clúster (Spark, PySpark).
-   Normativa y seguridad en almacenamiento de datos.

## Prácticas sugeridas

1.  **Comparativa SQL vs NoSQL**: Crear y consultar tablas en MySQL y MongoDB con un mismo dataset.
2.  **Ingesta desde API**: Ingestar datos desde una API pública hacia un DataFrame en pandas y guardarlos en MongoDB.
3.  **Procesamiento Distribuido**: Subir un dataset a HDFS y procesarlo con PySpark.
