## Contenidos

-   Python para ciencia de datos: pandas, NumPy, Matplotlib, Seaborn.
-   Análisis exploratorio y descriptivo.
-   Minería de datos: clustering, clasificación, regresión (Scikit-learn).
-   Validación y evaluación de modelos.
-   Preparación de *pipelines* de datos.
-   Documentación y automatización de procesos de análisis.

## Prácticas sugeridas

1.  **Análisis Exploratorio**: Análisis exploratorio de un dataset real (ej. Titanic o Iris) con pandas y Seaborn.
2.  **Modelado y Evaluación**: Entrenar un modelo de clasificación con Scikit-learn y evaluar métricas de precisión, recall y F1.
3.  **Creación de Pipeline**: Construir un *pipeline* de preprocesamiento y modelado en Scikit-learn.
