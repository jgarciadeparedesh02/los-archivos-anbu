## Contenidos

-   Aplicaciones interactivas de análisis con Dash/Streamlit.
-   Integración con APIs y bases de datos.
-   Automatización de reportes y paneles con Python.
-   Deploy de dashboards en la web o servidores.
-   Comparativa de librerías y frameworks de visualización avanzada.

## Prácticas sugeridas

1.  **Dashboard con API**: Construir un dashboard en Streamlit conectado a una API pública (ej. datos climáticos o financieros).
2.  **Automatización de Reportes**: Programar la actualización automática de un reporte diario en PDF con Python.
3.  **Despliegue de Dashboard**: Desplegar un dashboard en un servidor (ej. Heroku o similar) y evaluar su rendimiento.
