Para que puedas empezar a aplicar los conceptos que hemos visto, hemos preparado un pequeño conjunto de datos que simula las ventas en la cafetería del IES Ágora.

Puedes descargar el archivo CSV desde aquí:

[Descargar cafeteria_ventas.csv](/datasets/cafeteria_ventas.csv)

### Estructura del Dataset

El archivo `cafeteria_ventas.csv` tiene las siguientes columnas:

-   **Fecha**: La fecha y hora exactas de la transacción.
-   **Producto**: El nombre del producto vendido.
-   **Categoria**: La categoría a la que pertenece el producto (Bebidas, Bocadillos, Bollería).
-   **Precio**: El precio unitario del producto.
-   **Cantidad**: El número de unidades vendidas de ese producto en la transacción.
-   **ID_Transaccion**: Un identificador único para cada transacción de venta.

### Ideas para Empezar a Analizar

Con este dataset, podrías intentar responder a preguntas como:

1.  ¿Cuál es el producto más vendido?
2.  ¿A qué horas del día se producen más ventas?
3.  ¿Cuál es el ticket medio (gasto por transacción)?
4.  ¿Qué productos se suelen comprar juntos?

¡Anímate a explorar los datos con las herramientas que prefieras (Excel, Google Sheets, o incluso un pequeño script de Python con Pandas) y comparte tus hallazgos!
