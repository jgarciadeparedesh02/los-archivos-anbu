## Contenidos

-   Extracción de información desde datasets complejos.
-   Preprocesamiento y *feature engineering* con pandas, NumPy y Scikit-learn.
-   Introducción al análisis en tiempo real con PySpark Streaming.
-   Algoritmos de búsqueda y optimización para resolución de problemas.

## Prácticas sugeridas

1.  **Feature Engineering**: Realizar *feature engineering* sobre un dataset (creación de variables derivadas).
2.  **Análisis en Tiempo Real**: Implementar un flujo de análisis en tiempo real con PySpark Streaming sobre datos simulados.
3.  **Optimización**: Resolver un problema simple de optimización (ej. ruta más corta) usando algoritmos de búsqueda.
