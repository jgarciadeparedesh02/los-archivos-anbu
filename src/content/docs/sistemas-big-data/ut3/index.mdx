## Contenidos

-   Ciclo de vida del dato: recolección, limpieza, transformación, análisis y despliegue.
-   Estrategias de integración de datos.
-   Gobierno del dato: calidad, trazabilidad y consistencia.
-   Organización de proyectos Big Data con Python.

## Prácticas sugeridas

1.  **Limpieza de Datos**: Limpieza y normalización de un dataset con pandas (detección de valores nulos, duplicados, etc.).
2.  **Documentación del Ciclo de Vida**: Documentar el ciclo de vida de un dataset desde su recolección hasta el despliegue.
3.  **Integración de Fuentes**: Implementar un script en Python que combine datos de varias fuentes en un único DataFrame consistente.
