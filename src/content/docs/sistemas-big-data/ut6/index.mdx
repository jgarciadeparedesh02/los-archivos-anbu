## Contenidos

-   Fundamentos de visualización y BI.
-   Visualización con Python: Matplotlib, Seaborn, Plotly, Bokeh.
-   Dashboards interactivos con Dash o Streamlit.
-   Selección de métricas y KPIs.
-   Comunicación de resultados de manera clara y efectiva.

## Prácticas sugeridas

1.  **Generación de Gráficos**: Generar gráficos comparativos (barras, dispersión, líneas) con Matplotlib y Seaborn.
2.  **Dashboard Básico**: Construir un dashboard básico en Streamlit para explorar un dataset.
3.  **Diseño de KPIs**: Diseñar indicadores (KPIs) a partir de un conjunto de datos financieros o de negocio.
