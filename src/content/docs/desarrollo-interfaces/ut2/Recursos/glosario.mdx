## Términos clave de la UT2

### A

- **API (Interfaz de Programación de Aplicaciones)**: Conjunto de definiciones y protocolos que permite que dos componentes de software se comuniquen entre sí.

### D

- **DOM (Modelo de Objetos del Documento)**: Una representación en árbol de un documento HTML o XML que permite a los scripts (como JavaScript) interactuar y modificar la estructura, el estilo y el contenido de la página.
- **Declarativo**: Paradigma de programación donde se describe el resultado deseado sin detallar los pasos para alcanzarlo. En interfaces, se define "qué" se quiere mostrar, no "cómo".

### I

- **Imperativo**: Paradigma de programación que se basa en describir paso a paso cómo alcanzar un resultado. En interfaces, implica manipular directamente el DOM para cambiar la vista.

### J

- **JavaScript**: Lenguaje de programación interpretado, orientado a objetos y basado en prototipos, que se utiliza principalmente en el lado del cliente para crear páginas web dinámicas e interactivas.

### X

- **XML (Lenguaje de Marcas Extensible)**: Un lenguaje de marcado que define un conjunto de reglas para codificar documentos en un formato que es legible tanto por humanos como por máquinas.
