En esta Unidad de Trabajo (UT2), exploraremos el fascinante mundo de las **interfaces declarativas**, centrándonos en cómo lenguajes como XML han sentado las bases para el desarrollo de interfaces de usuario modernas. Aunque XML fue pionero en este campo, veremos cómo los frameworks actuales, como Angular junto con Ionic, han evolucionado para utilizar un enfoque similar basado en HTML declarativo con atributos personalizados.

### **Objetivos de la Unidad**

Al finalizar esta unidad, serás capaz de:

*   Comprender los principios fundamentales de las interfaces declarativas y su evolución.
*   Analizar la estructura y sintaxis de lenguajes de descripción de interfaces basados en XML.
*   Identificar las herramientas y entornos de desarrollo para la creación de interfaces declarativas.
*   Aplicar los conceptos de controles, propiedades y eventos en el diseño de interfaces.
*   Entender el proceso de edición, mantenimiento y generación de código multiplataforma.

### **Contenidos Clave**

1.  **Lenguajes de Descripción de Interfaces (XML y derivados):** Un viaje desde los orígenes de XML en la descripción de interfaces hasta su influencia en tecnologías actuales.
2.  **Elementos Básicos de una Interfaz Declarativa:** Componentes, atributos y la jerarquía visual.
3.  **Herramientas de Creación y Entornos de Desarrollo:** Exploración de IDEs y constructores visuales.
4.  **Controles y Propiedades en Frameworks Modernos (Angular-Ionic):** Cómo los componentes de UI se definen y personalizan.
5.  **Eventos y Controladores:** La interacción del usuario con la interfaz.
6.  **Edición y Mantenimiento de Interfaces:** Buenas prácticas y refactorización.
7.  **Generación de Código Multiplataforma:** El poder de un único código base para diferentes plataformas.

Esta unidad te proporcionará una base sólida para diseñar y desarrollar interfaces de usuario eficientes y atractivas, preparándote para las complejidades del desarrollo de aplicaciones modernas.
