## Tarea a entregar

Para completar esta unidad de trabajo, debes entregar el **documento de definición técnica (DDT)**.

Este documento debe contener:

1.  La elección del stack tecnológico (frontend, backend, BBDD) con su justificación.
2.  Un diagrama de la arquitectura de software del proyecto.
3.  El diseño del modelo de datos (esquema de la BBDD).
4.  La configuración del entorno de desarrollo y las convenciones de código.
