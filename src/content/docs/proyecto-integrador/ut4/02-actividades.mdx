## Tarea a entregar

El entregable principal de esta unidad es el **código fuente del proyecto**, que debe estar disponible en un repositorio de Git (GitHub, GitLab).

Se evaluará:

1.  La implementación de un conjunto significativo de las funcionalidades planificadas (MVP).
2.  La calidad y organización del código.
3.  El uso correcto del control de versiones (ramas, commits, Pull Requests).
4.  El seguimiento de una metodología ágil (evidenciado en el tablero de proyecto).
