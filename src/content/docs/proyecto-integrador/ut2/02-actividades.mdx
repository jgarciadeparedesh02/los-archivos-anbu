## Tarea a entregar

Para completar esta unidad de trabajo, debes entregar el **documento de diseño de producto**.

Este documento debe contener:

1.  La lista de requisitos funcionales de la aplicación.
2.  Los diagramas de flujos de usuario (user flows).
3.  Los wireframes de todas las pantallas.
4.  Un enlace al prototipo interactivo navegable en Figma o una herramienta similar.
