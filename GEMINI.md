## Mermaid
Cuando trates o crees diagramas, usa mermaid, pero no utilices paréntesis ya que dan error al interpretarlos.

## Tips

Cuando utilices :::tip o algún formato similar, utiliza el formato así:

:::note Importante para el navegador

Recuerda que, si vas a usar estos módulos en el navegador, tu HTML debe cargar el archivo JavaScript compilado (`dist/main.js`) como un módulo:
```html
<script type="module" src="dist/main.js"></script>
```
:::

## Icons

Para el uso de iconos en el proyecto utiliza sólo los que aparecen en la documentación de starlight. https://starlight.astro.build/reference/icons/#use-icons
